# 软件开发工具实践实验报告站

本网站用于展示“软件开发工具实践”课程的实验过程记录。网站采用静态站点形式构建，主要内容包括 Web 开发环境、代码编辑与管理、WSL2 Linux 环境、Linux 基础操作、用户与权限、环境变量、SSH 远程管理以及 Nginx 网站部署等实验内容。

## 项目目标

- 使用 VS Code 和 Node.js 搭建 Web 开发环境。
- 使用 Vite 创建静态网站项目。
- 使用 Git 完成代码版本管理并推送到 GitHub。
- 使用 WSL2 Linux 模拟 Web 服务器环境。
- 配置 Linux 用户、权限、环境变量和 SSH 服务。
- 使用 Nginx 部署静态网站并完成浏览器访问测试。

## 成果说明

本项目最终提交内容包括项目源码、GitHub 仓库、部署截图、实验报告页面和小组分工说明。每个实验步骤页面都保留关键命令、操作说明和截图位置，便于后续整理正式实验报告。
